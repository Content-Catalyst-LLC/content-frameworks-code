# Assumptions and Limitations

This scaffold uses synthetic data and simplified pathway metrics.

Limitations include:

- no live CMS connection
- no live WordPress crawl
- no automatic semantic validation of link quality
- no automatic reader-behavior analytics
- simplified transition coverage
- no replacement for editorial judgment
- no guarantee that a computationally connected path is conceptually useful

Production use should include CMS exports, analytics, human review, accessibility checks, and source-quality review.
