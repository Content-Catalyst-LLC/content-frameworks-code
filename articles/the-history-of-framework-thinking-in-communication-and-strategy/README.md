# The History of Framework Thinking in Communication and Strategy

This companion directory supports the article **"The History of Framework Thinking in Communication and Strategy."**

The article traces framework thinking across rhetoric, logic, classification, education, scientific modeling, strategy, management, advertising, information architecture, content strategy, knowledge management, digital publishing, and AI-assisted editorial systems.

## Core Claim

Framework thinking is not a modern content shortcut. It is a long historical practice for making complex knowledge usable through structured speech, memory, classification, learning, explanation, decision-making, navigation, governance, and institutional maintenance.

## Catalyst Canvas-Style Professional Workflow Layer

The workflows in this folder treat framework history as an auditable content-system capability:

- historical framework records
- lineage classification
- influence graph diagnostics
- cross-domain transfer-risk review
- governance readiness scoring
- risk and misuse review
- metadata completeness diagnostics
- article-map coverage summaries
- taxonomy coverage summaries
- internal-link diagnostics
- governance review queue generation
- Catalyst Canvas-style catalog exports
- reproducible CSV, JSON, Markdown, SQL, Java, Python, and R outputs

## Repository Structure

- `config/` — historical lineage rules and governance thresholds
- `html/` — GitHub block, footer navigation, method-wrapper snippets, and article components
- `css/` — reusable styling patterns
- `php/` — WordPress-oriented helper snippets
- `java/` — framework-history model and validation logic
- `python/` — Catalyst Canvas-style framework-history audit engine
- `r/` — historical period analysis, lineage comparison, and risk summaries
- `sql/` — schema and analytical queries
- `docs/` — article notes, modeling principles, governance documentation, and product notes
- `data/` — synthetic datasets
- `outputs/` — generated outputs
- `notebooks/` — notebook placeholders

## Quick Start

From this article directory:

```bash
python3 python/run_all_framework_history_workflows.py
Rscript r/run_all_framework_history_workflows.R
javac java/FrameworkHistoryModel.java && java -cp java FrameworkHistoryModel
sqlite3 outputs/framework_history.sqlite < sql/framework_history_schema.sql
sqlite3 -header -csv outputs/framework_history.sqlite < sql/framework_history_queries.sql > outputs/tables/sql_framework_history_report.csv
```
