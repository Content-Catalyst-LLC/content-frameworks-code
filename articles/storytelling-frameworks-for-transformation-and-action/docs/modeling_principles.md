# Modeling Principles

1. Storytelling frameworks organize transformation, action, meaning, and sequence.
2. Story is structure, not decoration.
3. Context should make starting conditions visible.
4. Actor mapping should preserve agency and responsibility.
5. Tension should be truthful, proportionate, and evidence-aware.
6. Sequence should show how movement happens.
7. Transformation claims should be bounded, supported, and mechanism-aware.
8. Action should match audience readiness and preserve choice.
9. Anecdotes should not substitute for evidence.
10. Governance should track representation, consent or source context, evidence support, review dates, and revision queues.
