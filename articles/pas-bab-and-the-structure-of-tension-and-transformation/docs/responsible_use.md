# Responsible Use

Use these workflows to identify PAS/BAB sequence gaps, weak current-state descriptions, disproportionate agitation, unsupported transformation claims, vague bridge mechanisms, and ethical-risk flags.

Do not use PAS/BAB readiness scores as proof that a message is ethical, effective, accessible, or trustworthy. The outputs support review; they do not replace audience research, evidence review, accessibility testing, or editorial judgment.
