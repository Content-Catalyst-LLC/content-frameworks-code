# Responsible Use

These workflows are editorial support tools.

Use them to:

- identify missing metadata
- detect footer-navigation mismatches
- check repository alignment
- review image and accessibility metadata
- validate controlled vocabularies
- inspect review-cycle status
- generate governance queues
- export content-system catalogs

Do not use metadata scores as proof of content quality, authority, accessibility compliance, or factual correctness. Metadata makes the system inspectable; human editorial review remains necessary.
