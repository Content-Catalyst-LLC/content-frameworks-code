# Interdisciplinary Frameworks and Knowledge Bridges

This companion directory supports the article **Interdisciplinary Frameworks and Knowledge Bridges**.

The workflows model interdisciplinary knowledge bridges as governed cross-domain structures: domain inventories, concept bridges, vocabulary alignment, evidence compatibility, method transparency, audience support, misuse warnings, governance queues, and catalog exports.

## Repository Structure

- `config/` — scoring rules and interdisciplinary bridge configuration
- `html/` — article components, GitHub block, footer navigation, method wrapper
- `css/` — reusable styling patterns
- `php/` — WordPress-oriented snippets
- `java/` — bridge-readiness scoring example
- `python/` — professional interdisciplinary audit workflow
- `r` — bridge readiness and domain-balance summaries
- `sql/` — schema and reporting queries
- `docs/` — article notes and governance notes
- `data/` — synthetic datasets
- `outputs/` — generated outputs
- `notebooks/` — notebook placeholders

## Quick Start

```bash
python3 python/run_all_interdisciplinary_bridge_workflows.py
Rscript r/run_all_interdisciplinary_bridge_workflows.R
javac java/InterdisciplinaryBridgeAudit.java && java -cp java InterdisciplinaryBridgeAudit
sqlite3 outputs/interdisciplinary_bridges.sqlite < sql/interdisciplinary_bridge_schema.sql
sqlite3 -header -csv outputs/interdisciplinary_bridges.sqlite < sql/interdisciplinary_bridge_queries.sql > outputs/tables/sql_interdisciplinary_bridge_report.csv
```
