# Assumptions and Limitations

This scaffold uses synthetic data and simplified taxonomy metrics.

Limitations include:

- no live WordPress taxonomy export
- no live CMS field validation
- no automatic semantic category validation
- no automatic reader-language testing
- no live search analytics
- simplified category balance scoring
- no replacement for editorial judgment

Production use should include CMS exports, content inventories, stakeholder review, user testing, search logs, and human taxonomy governance.
