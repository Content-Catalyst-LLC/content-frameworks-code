# Responsible Use

The workflows in this directory support taxonomy review and editorial governance. They do not replace human judgment.

Use them to:

- identify missing primary categories
- detect multiple-primary-category conflicts
- review tag sprawl
- detect thin or overloaded categories
- inspect taxonomy metadata readiness
- support category governance queues

Do not treat taxonomy metrics as final editorial decisions. Category fit, boundary logic, reader usefulness, and domain accuracy require human review.
