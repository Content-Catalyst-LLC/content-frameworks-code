# Assumptions and Limitations

The workflows use synthetic datasets and simplified scoring logic.

Limitations include:

- synthetic content inventory
- simplified evidence scoring
- simplified accessibility scoring
- simplified duplicate-risk signals
- no live WordPress API connection
- no automated page crawling
- no automatic broken-link HTTP checks
- no direct citation-quality verification
- no user-behavior analytics
- no replacement for editorial review

Production use requires real CMS exports, source review, accessibility testing, URL validation, editorial governance, and ongoing maintenance.
