# Responsible Use

These workflows are editorial intelligence tools.

Use them to:

- identify metadata gaps
- inspect article-map coverage
- evaluate link health
- find stale content
- flag weak evidence
- identify duplication risks
- inspect accessibility readiness
- generate governance queues
- support content maintenance decisions
- export Catalyst Canvas-style catalogs

Do not use them to automatically remove content, mechanically score intellectual quality, replace human editorial review, or treat metrics as proof of authority.
