# Modeling Principles

1. Hierarchy-of-effects models describe staged communication response; they do not predict behavior with certainty.
2. Awareness is not the same as understanding.
3. Understanding is not the same as trust.
4. Trust is not the same as preference.
5. Preference is not the same as conviction.
6. Conviction is not the same as action.
7. Action is not the same as adoption or follow-through.
8. Metrics are signals, not proof of response quality.
9. Response-stage labels should match content purpose and audience readiness.
10. Ethical review should check pressure, false urgency, overclaiming, hidden tradeoffs, and agency reduction.
