# Responsible Use

The workflows in this directory support editorial and system review. They do not replace human judgment.

Use them to:

- identify metadata gaps
- detect weak internal-link infrastructure
- review taxonomy coverage
- check repository readiness
- identify content needing governance review
- support AI-assisted audit workflows with human validation

Do not treat a high system-health score as proof of quality. Human review is still required for evidence, fairness, clarity, and domain fit.
