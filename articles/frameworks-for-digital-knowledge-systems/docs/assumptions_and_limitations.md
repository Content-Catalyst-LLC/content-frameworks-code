# Assumptions and Limitations

This scaffold uses synthetic data and simplified digital knowledge-system metrics.

Limitations include:

- no live WordPress or CMS crawl
- no live search-index analysis
- no automatic link-quality semantic validation
- no automatic source-quality review
- no live accessibility scan
- no live repository API inspection
- simplified system-health score
- no replacement for editorial judgment

Production use should include CMS exports, crawl validation, analytics, accessibility testing, source review, and human governance.
